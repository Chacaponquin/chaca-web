# YAML

You can export the data generated by schemas or any type of data to [yaml](https://es.wikipedia.org/wiki/YAML) format

```js
const userSchema = chaca.schema({
   id: schemas.id.uuid(),
   username: schemas.internet.userName(),
   image: { type: schemas.image.people(), posibleNull: 50 },
   posts: {
      type: schemas.id.uuid(),
      isArray: {
         min: 1,
         max: 5,
      },
   },
})

await userSchema.generateAndExport({
   filaName: 'data',
   format: 'yaml',
   location: 'folder',
})
```
