# Javascript

You can export the data generated by modules or any type of data to [javascript](https://developer.mozilla.org/es/docs/Web/JavaScript) format

```js
const userSchema = chaca.schema({
  id: modules.id.uuid(),
  username: modules.internet.userName(),
  image: { type: modules.image.people(), posibleNull: 50 },
  posts: {
    type: modules.id.uuid(),
    isArray: {
      min: 1,
      max: 5,
    },
  },
})

await userSchema.generateAndExport({
  filaName: "data",
  format: "javascript",
  location: "folder",
})
```
