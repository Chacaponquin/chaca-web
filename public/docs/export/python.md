# Python

You can export the data generated by modules or any type of data to [python](https://www.python.org/) format

```js
const userSchema = chaca.schema({
  id: modules.id.uuid(),
  username: modules.internet.username(),
  image: { type: modules.image.people(), posibleNull: 50 },
  posts: {
    type: modules.id.uuid(),
    isArray: {
      min: 1,
      max: 5,
    },
  },
})

await userSchema.generateAndExport({
  filaName: "data",
  format: "python",
  location: "folder",
})
```
